Steps to Follow to run to this program

1. Copy this file or gaussian_2d folder to any of the directory
2. Go to that gaussian_2d folder
3. Fire the following command to compile the code
	gcc -o Task4 Task4.c
4. Fire the following command to execute the code
	./Task4
5. it will ask for file path
	Provide the file path and name.
6. Provide the absolute path
	Ex: E://xyz//asd.txt
or Provide the relative path, but you have to place the file ads.txt in gaussian_2d folder
	asd.txt  
7. Press Enter to execute the program
